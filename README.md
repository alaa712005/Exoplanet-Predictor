# 🪐 Exoplanet Prediction Studio

Welcome to the **Exoplanet Prediction Studio** project! This repository provides an advanced interactive web application and machine learning inference engine designed to analyze telescope transit signals and predict exoplanets.

## 🚀 Live Demo

You can try the live application here: [[Exoplanet Prediction Studio App](https://exoplanet-predictor-2ehx58wbv8qsdwadbstq3r.streamlit.app)](https://exoplanet-predictor-2ehx58wbv8qsdwadbstq3r.streamlit.app/)

## 🎯 Project Objective

In the search for distant worlds, accurately classifying telescope transit data to distinguish true exoplanets from false positives or background noise is critical. This project leverages machine learning models combined with an interactive **Streamlit** dashboard to serve as an intelligent inference and exploration platform.

## 📊 Features & Capabilities

* **Interactive Prediction Studio:** Test custom telescope parameters or load preset scenarios (Confirmed Planets vs. False Positives/Noise).
* **EDA Dashboard:** Explore data distributions and correlations visually using Plotly.
* **Model Performance Insights:** Evaluate metrics and performance indicators of the classification models.
* **Advanced UI/UX:** Built with customizable theme options (Dark/Light mode support).

## ⚙️ Tech Stack

* **Frontend/UI:** Streamlit, Plotly
* **Machine Learning & Data:** Scikit-Learn, Pandas, NumPy, Joblib
* **Deployment:** Streamlit Cloud & GitHub

## 📚 References & Data

* **Domain:** Astronomy & Exoplanet Transit Detection Data Pipelines.
